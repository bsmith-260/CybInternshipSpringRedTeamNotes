## Hak5 Payload Studio Pro

`Payload Studio Pro` is an application created by `Hak5` for the configuration of their penetration testing tools. Payload Studio Pro offers a command line setup for writing scripts for their available devices. 

Payload Studio Pro offers script configuration through DuckyScript for the following devices:
1. USB Rubber Ducky
2. Bash Bunny
3. Key Croc
4. Shark Jack
5. Packet Squirrel
6. LAN Turtle
7. O.MG

## How to Download

Payload Studio Pro can be purchased directly through the Hak5 website, and offers full access to the command line where you can configure a payload for the specific Hak5 device you are using. The total cost of the service is $60.

![[Pasted image 20260312182728.png]]

After purchasing the Pro subscription, you can access it through the link, [Hak5 PayloadStudio](https://payloadstudio.com/login/) using the email you purchased with, and the license key sent in the purchase confirmation email.

![[Pasted image 20260312182857.png]]

Once you login with your information, you will be prompted to select what device you are trying to write a payload for. Select the device you are using.

![[Pasted image 20260313144157.png]]

Each selection has differences, but all use DuckyScript. In this example, we are using the USB Rubber Ducky, and can see the default `Hello, World!` payload that Hak5 gives us. This payload will result in the string "Hello, World!" being written in a text editor if you have one open on your device.

![[Pasted image 20260313144459.png]]

The script for the payload can be changed and the options that can used can be found on the [[DuckyScript]] information page.


## How to Use

After purchasing `Hak5 Payload Studio`, you are able to login with your email address and the given license key through your linked email. 

Select the device configuration of the tool that you are using, and the script editor will then appear.

A default comment for the Payload title, author, and description will be listed, with a default 'Hello World!' string. 

You can edit the payload for the `USB Rubber Ducky` with the `DuckyScript` scripting language.
